# README

仲間で楽しむ順位予想アプリ「いろいろ順位」

サービス概要
* 「名前」ではなく「色」で識別する、直感的な順位予想共有アプリです。
* 1つの端末で最大4人まで、スポーツや賞レースの順位をパレットから色を選んで素早く入力できます。
* シーズン後に正解の色の並びを入れるだけで、誰が最も近かったかを一瞬で判定し、その場の盛り上がりを加速させます。


このサービスへの思い・作りたい理由
毎年、友人たちとプロ野球のシーズン開幕前に順位予想を楽しんでいます。しかし、シーズンが終わった後に誰の予想が当たっていたかを照らし合わせる作業が意外と面倒で、手計算やメモの確認に時間が取られてしまいます。 せっかく集まってお酒を飲んだり盛り上がったりしているのに、「集計という作業」に時間を奪われるのがもったいないと感じたのがきっかけです。 「オレンジ＝ジャイアンツ」「青＝中日」のように、ファンなら一目でわかる「色」をUIの主役に据えることで、入力の煩わしさを排除し、視覚的に「誰が一番正解に近いか」をパッと見で分かち合える体験を作りたいと考えています。

ユーザー層について
* 毎年恒例で順位予想イベントを行っている友人グループ
    * 理由：既存の集計の手間を解消したいというニーズが最も強いため。
* プロ野球やJリーグ、M-1グランプリなどのファン
    * 理由：チームカラーやイメージカラーが定着しており、「色」による直感的な操作と相性が良いため。
* 「その場」のコミュニケーションを重視する人
    * 理由：URL共有の手間すら省き、1つのスマホを回してその場で完結させたいという需要に応えるため。

サービスの利用イメージ
1. 予想時（シーズン前）：1人がアプリを開き、その場にいる4人がスクロールしながら自分の予想順位に色を置いていきます。データは端末（またはDB）に保存されます。
2. 結果確認（シーズン後）：再び集まった際に、実際の順位（色の並び）を入力します。
3. 価値：瞬時に全員のスコアが計算され、色鮮やかな比較チャートが表示されます。「お前、ここだけ当たってるじゃん！」「この色の並び、全然違う！」といった会話が即座に始まり、集計作業なしで最高の結果発表イベントを楽しめます。


ユーザーの獲得について
* SNS（X/Twitter）での発信：プロ野球の開幕前やM-1などのイベント直前に、色の並びの美しさをキャプチャで投稿し、視覚的な面白さをアピールします。
* ハッシュタグ活用：特定のスポーツやイベントのハッシュタグを使い、予想を楽しんでいるコミュニティにリーチします。
* 口コミ：1つの端末で完結するため、1人が使えばその場にいる4人に確実に体験が伝わり、横展開を狙います。
サービスの差別化ポイント・推しポイント
* 「色」特化のUI：テキスト入力を極限まで減らし、色を選ぶだけで完成する「感覚的な心地よさ」を追求しています。
* ユニバーサルな汎用性：特定のスポーツ専用アプリではないため、ユーザーが自由に色を定義することで、競馬、アイドル、社内行事など、あらゆる「順位」に応用可能です。
* オフライン・リアル重視：URLを共有して各自で入力させるのではなく、「1つの端末を囲んでワイワイする」というアナログな良さをデジタルで加速させる点にこだわっています。


機能候補
MVPリリース時
* イベント作成機能：予想するタイトル（例：2026年セ・リーグ）を登録。
* カスタムカラーパレット作成：<input type="color"> を使い、そのイベントで使用する色を自由に追加。
* 4人分の予想入力機能：縦スクロールで4名分、パレットから色をタップして順位を埋める。
* 正解入力・判定ロジック：正解の並びを入力し、一致数やニアピン度を算出・表示する機能。


本リリースまでに追加したい機能
* ドラッグ＆ドロップによる並び替え：SortableJSなどを用いた直感的なUI。
* 結果画面の画像生成・シェア機能：SNSに投稿しやすい比較画像の自動作成。
* 過去ログ保存：自分の歴代の予想的中率をグラフ化。
* テンプレート機能：プロ野球12球団などの色セットをワンタップで呼び出し。


使用する技術スタック
実装イメージ
* スコア算出ロジック： 正解配列と予想配列を比較し、indexが完全一致なら3点、前後1位のズレなら1点、といった配点ロジックをRuby側で実装します。
* 色選択UI： JavaScriptで、パレットから選んだ色のHEXコードをhidden_fieldにセットする、またはReact等のコンポーネントで動的に反映させます。


技術構成
* フレームワーク：Ruby on Rails 7系
* データベース：PostgreSQL
* デプロイ先：Renderなど無料のものを選択
* 使用予定のライブラリ：
    * Tailwind CSS：色の表示を綺麗にレイアウトするため。
