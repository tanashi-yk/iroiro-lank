# iro-lank (イロランク)
「いろいろ順位」
仲間で楽しむ「色」による順位予想共有アプリ

## サービス概要
* **「名前」ではなく「色」で識別する**、直感的な順位予想共有アプリです。
* 1つの端末で最大4人まで、スポーツや賞レースの順位をパレットから色を選んで素早く入力できます。
* シーズン後に正解の色の並びを入れるだけで、誰が最も近かったかを一瞬で判定し、その場の盛り上がりを加速させます。

## このサービスへの思い・作りたい理由
毎年、友人たちとプロ野球のシーズン開幕前に順位予想を楽しんでいますが、シーズン後の「答え合わせ（集計作業）」に時間がかかり、せっかくの盛り上がりが途切れてしまうのがもったいないと感じていました。
「オレンジ＝ジャイアンツ」「青＝中日」のように、ファンなら一目でわかる「色」をUIの主役に据えることで、入力の煩わしさを排除。視覚的に「誰が一番正解に近いか」をパッと見で分かち合える体験を作りたいと考えました。

## ユーザー層について
* **野球好きのコミュニティメンバー**
  * 理由：日常的に順位予想イベントを行っており、既存の集計の手間を解消したいというニーズが強いため。
* **「その場」のコミュニケーションを重視するグループ**
  * 理由：URL共有の手間すら省き、1つのスマホを回してその場で完結させたいという需要に応えるため。

## サービスの利用イメージ
1. **予想時（シーズン前）**: 1人がアプリを開き、その場にいる4人がスクロールしながら自分の予想順位に色を置いていきます。
2. **結果確認（シーズン後）**: 再び集まった際に、実際の順位（色の並び）を入力します。
3. **価値**: 瞬時に全員の予想順位と正答のインデックスと色を比較し正答率を確かめます。（何個インデックスの色があってるかだけ分かれば結果として見えます）すごく伊sんぷるな設計なので「お前、ここだけ当たってるじゃん！」といった会話が即座に始まり、集計作業なしで最高の結果発表イベントを楽しめます。

## サービスにかけるこだわり
本サービスは「その場の盛り上がりを加速させるシンプルさ」を追求しています。多機能化によってこの「手軽さ・操作性」が失われる場合はあえて実装しないなど、ユーザー体験のシンプルさを最優先に開発します。まずは身近なコミュニティや家族に使ってもらい、直接フィードバックを得ながら磨き上げます。

## 改善について
リリース時に仲間内でペナントを想定したテストを行なって改善点を集約。
またリリース後はお笑い賞レースや音楽チャートなど日常的に順位予想できるチャンスを伺って知見を集め
都度、よりユーザビリティを意識した改修を行う。

## 機能候補
### MVPリリース時
* **イベント作成機能**: 予想するタイトル（例：2026年セ・リーグ）を登録。
* **4人分の予想入力機能**: 縦スクロールで4名分、パレットから色をタップして順位を埋める。
* **正解入力・判定ロジック**: 正解の並びを入力し、一致数やニアピン度を算出・表示する機能。


### 本リリース時
* **URL発行・共有でデータ維持**:予想ごとにIDが含まれたURLを発行、共有して過去のデータを確認できるように 
* **人数の追加**:基本は４人ゲームですが本リリース後に人数追加機能を実装

### 技術構成
* **フレームワーク**: Ruby on Rails

* **データベース**: PostgreSQL
