### 本サービスの概要（700文字以内）
 サービス目的・概要 本サービスは、「名前」ではなく「色」で識別する、直感的な順位予想共有アプリです。プロ野球やJリーグ、賞レースなど、チームカラーやイメージカラーが定着しているイベントにおいて、パレットから色を選ぶだけで素早く順位予想を入力・集計し、その場の盛り上がりを加速させることを目的としています。
 解決する課題 友人同士での順位予想イベントは楽しいものですが、シーズン終了後の結果照合や的中率の計算は意外と手間がかかります。せっかく集まって盛り上がっている最中に、手計算やメモの確認といった「集計作業」に時間を奪われるのはもったいないという実体験に基づいています。本サービスは、結果入力から勝者判定までを一瞬で完結させ、純粋に予想を楽しむ時間を提供します。
 想定ユーザー
* スポーツ・賞レース・アイドルファン： カラーが定着しているジャンルの熱心なファン層。
* 恒例の予想イベントを行うグループ： 既存の集計の手間を解消したいと感じている友人同士。
* 対面での交流を重視する人： URL共有の手間すら省き、1つの端末を回してその場で完結させたい層。
